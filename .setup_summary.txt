================================================================================
                          🚀 快速开始步骤
================================================================================

1. 安装必需依赖：
   sudo apt-get install libeigen3-dev libcgal-dev libgmp-dev libmpfr-dev libboost-all-dev

2. （可选）克隆 VCGlib：
   cd /workspace/third_party
   git clone https://github.com/cnr-isti-vclab/vcglib.git

3. 配置项目：
   mkdir build && cd build
   cmake ..

4. 查看配置摘要（会显示所有依赖和 AI 模型状态）

5. 编译项目：
   cmake --build . -j$(nproc)
